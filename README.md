## Features
- Encrypted user authorizaton
- Database initialization
- New user signup
- User login/logout
- User settings
- Modern user interface
- Bulma framework
- Limited custom css/js
- Easily customizable

## Setup
``` 
git clone https://github.com/mrtnnk/python-flaskex
cd Flaskex
pip install -r requirements.txt
python app.py
```
